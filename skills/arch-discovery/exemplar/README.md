# Exemplar: realized §1 in shape

This directory holds **read-before-author** examples of fully realized discovery sections. Setup Mode (step 8) routes the author here so they see the target shape before writing.

## Contents

- `worked-section.md` — a fully realized §1.1 / §1.2 / §1.3 trio on a generic 3-actor system. Demonstrates the **Narrative → Diagram → Claims** shape. Read this in full before authoring §1 of a new discovery.

## Why this exists

The skill's `template.md` describes the shape in prose; experience shows that prose-only instructions reliably produce bullet-only first drafts (the "axiom list" failure mode). Showing a realized example primes the author on tone, density, and the load-bearing role of each subsection — which prose alone cannot.

## Adding new exemplars

Add when a discovery doc reaches `Status: Discovery (reviewed)` and demonstrates a section shape worth canonicalizing (e.g. an unusually clear §4.3 packet walk, an unusually clean §5 lifecycle re-walk). Strip project-specific detail, keep the shape.
